#### Descripción


## Instalar librerias requeridas para el proyecto
con yarn
```
~$ yarn install
```
con npm
```
~$ npm install
```

### Compiles and hot-reloads for development
con yarn
```
~$ yarn serve
```
con npm
```
~$ npm run serve
```

### Compiles and minifies for production
con yarn
```
~$ yarn build
```
con npm
```
~$ npm run build
```

### Correr test de pruebas unitarias

con yarn
```
~$ yarn test:unit
```
con npm
```
~$ npm run test:unit
```

### Lints and fixes files
con yarn
```
~$ yarn lint
```
con npm
```
~$ npm run lint
```
